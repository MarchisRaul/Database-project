Pentru a putea vizualiza proiectul, este nevoie de programul "XAMPP".
Dupa ce se deschide Control Panel-ul programului, se da "Start" in dreptul modulelor "Apache" si "MySql".
Se deschide un browser web si se acceseaza link-ul: http://localhost/MainPage.html

Fisierul "Rezolvarea cerintelor" reprezinta solutiile pentru toate exercitiile propuse.
